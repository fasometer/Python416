Сайт состяния автоматичекий линий
1) Основная задача сайта хранить и дополнять базу данных проблем с автоматическими линиями.
Поиск по базе данных по номеру линии, по узлу линии по назваию проблемы и т.д.
Дополнение базы данных.
2) Создание задачи по неисправности линии 
	возможность заполнение задачи с телефона (фото).
3) распределение задач админом с рассылкой уведемления о задаче отделам
4) Доступ к техдокументации по линиям (схемы, файлы настройки)
